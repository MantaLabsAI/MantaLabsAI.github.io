# MantaLabsAI.github.io

Manta Labs marketing site — [mantalabs.ai](https://mantalabs.ai)

## Running locally

```
npx serve .
```

Then open `http://localhost:3000`.

## Stack

- Static HTML + React (CDN, no build step)
- Babel standalone for JSX transpilation
- Lucide icons
- Formspree for contact form

## Custom domain

CNAME configured for `mantalabs.ai`.
